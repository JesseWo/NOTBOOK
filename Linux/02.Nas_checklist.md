# 小爱音箱
### 1. 添加AirPlay 支持
[docker hub: airconnect](https://hub.docker.com/r/1activegeek/airconnect)

docker-compose
```bash
version: "3.8"

services:
  airconnect:
    image: 1activegeek/airconnect
    container_name: airconnect
    network_mode: host
    restart: unless-stopped
```
### 2. 添加 Spotify connect 支持
SpotConnect
[Github](https://github.com/philippe44/SpotConnect)
[Docker](https://github.com/GioF71/spotconnect-docker)

```yaml
# UPnP Mode
version: "3"

volumes:
  config:

services:
  spotconnect-upnp:
    image: giof71/spotconnect:latest
    container_name: spotconnect-upnp
    network_mode: host
    environment:
      - SPOTCONNECT_MODE=upnp
      - PUID=1000
      - PGID=1000
    volumes:
      - config:/config
    restart: unless-stopped



#AirPlay Mode
version: "3"

volumes:
  config:

services:
  spotconnect-airplay:
    image: giof71/spotconnect:latest
    container_name: spotconnect-airplay
    network_mode: host
    environment:
      - SPOTCONNECT_MODE=raop
      - PUID=1000
      - PGID=1000
    volumes:
      - config:/config
    restart: unless-stopped
```

# HomeAssistance
通过 casaos 安装 HA
### 安装HACS (Home Assistant Community Store)
进入 HA 的 docker container, cd 到 config 目录, 执行
```
wget -O - https://get.hacs.xyz | bash -
```
重启 HA docker container

依次点击
Settings-Devices & Services - Add Intergration
搜索 HACS, 绑定 Github
安装完成.

### 添加xiaomi设备
> 突破全球地区限制
比如有两个账户, 一个是中国大陆, 一个是新加坡账户, 能同时绑定多个. (用米家APP就只能登录一个)

依次点击
Settings-Devices & Services - Add Intergration
搜索 xiaomi, 绑定小米账户(有2FA验证, 需要发送验证码给你的手机号或者邮箱)

# Aria2
通过 casaos store 下载 aria2,
修改 rpc-token, 保存后 docker containter 自动重启
点击 aira2 进入 AriaNG 前端页面, 会弹出"认证失败", 不要慌,
点击左边菜单栏 AriaNG 设置(或者叫作"常规设置"), 可以修改RPC密钥, 或者添加其他 aria2 实例
